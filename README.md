# RG Auto Sales

Sistema completo para gestão e intermediação de compra e venda de viaturas.

A **RG Auto Sales** é uma plataforma que combina presença digital com um sistema administrativo interno para gerir carros, clientes, leads e vendas, com foco em eficiência, transparência e crescimento do negócio.

---

##  Funcionalidades

###  Frontend (Público)
- Listagem de carros disponíveis
- Página de detalhes dos veículos
- Formulário de agendamento de test drive
- Página para vender viatura
- Interface moderna e responsiva

###  Backend / Admin
- Gestão de carros (adicionar, editar, apagar)
- Gestão de leads e clientes
- Sistema de vendas
- Dashboard com KPIs
- Módulo financeiro (lucros, comissões)
- Upload e gestão de fotos
- Relatórios e exportação de dados
